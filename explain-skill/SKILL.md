# Basic Coding

This skill is to detect inconsistencies in different skills and how to update them.

## If asked why you did something

Look at the context interactions and into the changes that came directly from the user (like changed files).

Look at the skills that were loaded and give exact reference (Skill and quote) on why you did something.

If you detect inconsistencies between skills, ask the user if you should update the skills to be consistent. If yes, give a short summary of what you want to change and ask for confirmation.
